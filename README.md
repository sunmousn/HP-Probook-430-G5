* CPU：i5 7200U & Intel HD Graphics 620  
* 主板：HP 8377
* 声卡：Conexant CX8200  
* 蓝牙：Intel Bluetooth  
* 内存：Hynix DDR4 2133MHz 8G  
* 硬盘：NVMe SK hynix PC401 256G
